## install softwares in ubuntu

    $ sudo os/ubuntu.sh

### install softwares in centos7 as root

    $ os/centos7.sh

## install with init

    $ git clone https://github.com/yuweijun/bin.git bin.git
    $ cd bin.git
    $ sudo os/ubuntu.sh
    $ ./install.sh init

### reinstall

    $ ./install.sh
